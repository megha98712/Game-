# 🎮 Math Explorer - Enhanced Edition

**Math Explorer** is a fun, jungle-themed math adventure game designed to help players improve their math skills while having an exciting gameplay experience! 🌴🧮

## ⚡ Features

- 🌴 Jungle-themed math adventure
- 🟢 **Two difficulty levels**: Easy & Hard
- ⭐ Real-time **score tracking**
- ❤ **Lives system**: Lose a life on wrong answer
- ⏱ Countdown **timer** for each question
- 🎵 Background music & sound effects:
  - Correct answer: 🌟
  - Wrong answer: ❌
- 🖌 Smooth **animations** for UI elements
- 🎯 Game Over screen with **final score** & **high score**
- 🔊 Music toggle button
- ✅ Responsive **buttons** and **progress bar**

---

## 🛠 Technologies Used

- **HTML5** – UI structure
- **CSS3** – Styling & animations
- **JavaScript** – Game logic, questions, timer
- **LocalStorage** – Save top 10 high scores
- **Audio/Visual assets** – Music & jungle background

---

## 🚀 How to Run

1. Clone or download the repository into a folder.  
2. Make sure the following files exist in the folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `jungle1.jpg`
   - `sweet_song.mp3`

3. Open **index.html** using **Chrome** (recommended) via:
   - Right-click → **Open with → Chrome**
4. Enter your name in the login screen and click **Start ▶**.
5. Choose **Easy** or **Hard** mode to begin solving math questions.
6. Click the **🔊 music button** to toggle background music.

> **Note:** Music starts only after you click **Start** due to browser autoplay restrictions.  

---

## 🎮 Gameplay Instructions

- Type your answer in the input field. 📝
- Press **Enter** or click **✅ Check** to submit.
- ✅ Correct answers:
  - Increase your **score**  
  - Add extra **time**
- ❌ Wrong answers:
  - Decrease your **lives**
- ⏰ Game ends when **lives = 0** or **time runs out**.
- On Game Over, you can:
  - 🔁 Play Again
  - 🏡 Return to Menu

---

## 🔧 Customization

- Replace **jungle1.jpg** to change the background 🏞️  
- Replace **sweet_song.mp3** to change background music 🎵  
- Edit **script.js** to modify:
  - Difficulty levels
  - Question types (+, -, *, /, ^)
